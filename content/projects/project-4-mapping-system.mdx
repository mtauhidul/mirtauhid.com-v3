## Project Overview

The Interactive Web Mapping System is designed to allow users to analyze geospatial data interactively. The system supports data upload, real-time updates, and various visualization tools to explore data on a map.

### Features

- Geospatial data upload and processing.
- Interactive map with zoom, pan, and layers.
- Data-driven decision-making with customizable visualizations.
- Real-time data updates and overlays.

### Technologies Used

- **Frontend**: React, Leaflet.js for maps
- **Backend**: Node.js, Express
- **Other**: Mapbox API, Firebase for real-time updates

---

## Challenges Faced

- Handling large datasets and ensuring smooth performance on the front end.
- Optimizing map rendering for multiple layers and real-time updates.
- Ensuring cross-browser compatibility and responsiveness.
