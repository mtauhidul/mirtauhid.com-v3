## Project Overview

The Hospital Workflow Management System is a web-based solution designed to streamline the daily operations of a hospital. It enables seamless management of patient data, appointment scheduling, staff roles, and real-time room allocation.

### Features

- Real-time patient data monitoring.
- Role-based dashboards for doctors, nurses, and admin.
- Real-time room and resource allocation.
- Integration with electronic health record (EHR) systems.

### Technologies Used

- **Frontend**: React, Tailwind CSS
- **Backend**: Firebase Firestore
- **Other**: Google Cloud Functions, EHR APIs

---

## Challenges Faced

- Handling real-time data synchronization across multiple dashboards.
- Ensuring HIPAA compliance for medical data security.
- Designing intuitive user interfaces for medical staff.
